# PJ1デザイン評価 クイックテンプレート

## 使用方法

1. 本テンプレートをコピー
2. `[ ]` 部分を埋める
3. 判定に応じて不要なセクションを削除
4. mailbox/Notionへ貼り付け

---

## 判定基準 早見表

| 観点 | A（即アプローチ） | B（条件付き） | C（保留） | D（除外） |
|------|------------------|---------------|-----------|-----------|
| **ビジュアル** | プロ品質・統一感あり | 部分的に改善余地 | 全体的に古い/乱雑 | - |
| **レイアウト** | 導線明確・UX良好 | 一部混乱あり | 情報過多・迷いやすい | - |
| **CTA** | 明確・複数配置 | 1つだけ配置 | 弱い/最下部のみ | - |
| **モバイル** | 最適化済み | やや問題あり | 未対応/崩れあり | - |
| **信頼要素** | 実績・証明充実 | 最低限あり | 不足 | - |
| **連絡可否** | フォーム/メール可 | - | - | 連絡手段なし |

## 即修正案 例文集

### VISUAL系
- 「トップ画像を差し替え。プロの教室写真に変更」
- 「フォント統一。見出しはNoto Sans、本文は明朝に統一」
- 「配色を3色以内に統一。現在5色以上使用」

### LAYOUT系
- 「ファーストビューにCTAボタン追加。スクロール追従型」
- 「メニューをハンバーガーから通常メニューに変更」
- 「電話番号をヘッダーに固定表示」

### CONTENT系
- 「生徒の声を3件追加。実績より体験談を優先」
- 「講師紹介に顔写真を追加」
- 「アクセス情報をページ上部に配置」

### TECH系
- 「画像圧縮。表示速度2秒以内に改善」
- 「SSL対応。https化」
- 「モバイル表示崩れ修正」

## 次工程連携メモ

### nanamiへの返信時
- 本テンプレート全文をmailboxに貼り付け
- 判定理由と即修正1点は必須

### itadoriへの連携時（A/Bのみ）
- 候補名・URL・判定を要約
- 「即修正1点」を営業トークに活用可能か確認

### Notion登録時
- 「判定理由3点」→ メモ欄
- 「即修正1点」→ 次アクション欄
- 「次アクション推奨」→ ステータス更新
