# Blog to X Post Conversion Templates

ブログ記事や調査レポートをX投稿に変換するためのテンプレート集。
@yossyk14（個人アカウント）と@YSTConsulting（企業アカウント）向けに分類。

---

## 📝 変換フロー概要

```
ブログ記事/調査レポート → 切り取り → 投稿向け再構成 → ハッシュタグ追加 → 投稿
```

### 基本ルール
1. **一文一義**: 1つの投稿に1つの主張・情報
2. **前置きなし**: 「〜について書きました」は不要
3. **具体値入り**: 「効果が出る」→「反応率2.3倍」
4. **改行活用**: 3〜4行に分割して読みやすく

## 🔄 変換チェックリスト

### ブログ記事読み込み時の確認項目
- [ ] 記事の主題（テーマ）は何か？
- [ ] キーワード・具体値はあるか？
- [ ] 独自の視点・経験が含まれているか？
- [ ] 読者にとっての価値は何か？

### 投稿変換時の確認項目
- [ ] 140〜280文字に収まっているか？
- [ ] 一文が長くなりすぎていないか？
- [ ] ハッシュタグは2〜4個に絞れているか？
- [ ] 投稿目的（共感/情報/拡散）が明確か？
