# PJ1 Copy Generator

PJ1パイプラインにおける営業DMコピー生成・品質検証の標準化スキル。

## 生成タイプ一覧

| タイプ | 特徴 | 適用シナリオ |
|--------|------|-------------|
| **主張型** | EduShiftの強み・実績を前面に | 初回接触・信頼構築が目的 |
| **体験型** | 相手の悩み・体験に共感 | 共感ベースで関係構築 |
| **ニュース型** | 時事・季節ネタを絡めた情報提供 | タイムリーなアプローチ |

---

## 品質チェックリスト

### 【主張型】品質チェックリスト

**必須要素（全部入れる）**
- [ ] 「元栄光ゼミナール部長」または「元大手塾部長」が明記されている
- [ ] 「個人塾・フリーランス講師特化」というターゲット絞り込みがある
- [ ] 「生徒が増えるHP」という成果志向のキーワードが含まれている
- [ ] 「30分無料相談」など、具体的な次アクションが示されている
- [ ] 相手のURL/サイト名が本文に自然に含まれている

**構成チェック（順序確認）**
- [ ] 1文目で「同業者」または「教育業界経験者」としての信頼性が示されている
- [ ] 2-3文で共感ポイント（独立の悩み・生徒集めの苦労）が提示されている
- [ ] 中盤で相手サイトの具体的な強みが1点以上指摘されている
- [ ] 後半で改善ポイントが「具体的」かつ「1-2点」に絞られている
- [ ] 最後に「無理な営業はしません」または同義の緩和フレーズがある

**トーン・スタイル**
- [ ] 差し出す姿勢（「〜です」「〜ます」）が基本
- [ ] 親身なアドバイザー感があり、押し付けがましくない
- [ ] 「〜すべき」「〜するべき」などの強制表現が使われていない

**禁止表現チェック**
- [ ] 「無料モック」という表現が含まれていない
- [ ] 「HP制作します」という安売り表現が含まれていない
- [ ] 過度な自社アピール（「弊社は〜」「当社では〜」の連続）がない
- [ ] 「業界トップクラス」「No.1」などの根拠のない claim がない

---

### 【体験型】品質チェックリスト

**必須要素（全部入れる）**
- [ ] 相手の「悩み」または「苦労」が具体的に言語化されている
- [ ] 「HPがあるだけでは不十分」という教育を含んでいる
- [ ] 親（保護者）の視点を意識した表現が含まれている
- [ ] 過去の成果事例（問い合わせ数増加など）が具体的数字で示されている
- [ ] 相手の強みを言及しつつ、親への伝わり方を指摘している

**構成チェック（順序確認）**
- [ ] オープニングで共通の悩みを提示（「〜と感じていませんか？」）
- [ ] 相手の現状を「褒め」で入る（強みを認めてから指摘）
- [ ] 「親が見たい情報」や「保護者目線」というフレーズが含まれている
- [ ] 具体的な成果数字（月1件→月4件など）が提示されている
- [ ] CTAが「無料相談」で、ハードル感が低い言い回しになっている

**トーン・スタイル**
- [ ] 「〜かもしれません」「〜なのでは」などの推量表現が適度に使われている
- [ ] 相手の苦労を「わかる」という共感表現がある
- [ ] 丁寧語＋親しみやすさのバランスが取れている

**禁止表現チェック**
- [ ] 「間違っています」「〜すべきです」などの断定・強制表現がない
- [ ] 「うちに任せれば」などの受注前提の表現がない
- [ ] 相手の現状を「ダメ」「悪い」と否定的に言い切っていない

---

### 【ニュース型】品質チェックリスト

**必須要素（全部入れる）**
- [ ] 時事・季節・業界トレンドなどの「タイムリーなネタ」が入っている
- [ ] 「春の募集」「新学期」「受験シーズン」などの時期感が明確
- [ ] 具体的な数字やデータ（「平均4.2校のHPを比較」など）が含まれている
- [ ] 緊急性・タイムリミットが「押し付けすぎない」範囲で示されている
- [ ] 今すぐ実施できる具体的アクション（チェックリストなど）が含まれている

**構成チェック（順序確認）**
- [ ] 1文目で季節・時事ネタが示され、注目を引く
- [ ] 「実は〜」という裏情報・業界データが提示されている
- [ ] 相手サイトの現状を「もったいない」と評価している
- [ ] 「今すぐできる改善」が3点以内に絞られてリスト化されている
- [ ] 時期に応じた特別プラン・特急対応の案内がある

**トーン・スタイル**
- [ ] 「〜ですね」「〜かもしれません」などの共感・推量表現がある
- [ ] 情報提供メインで、営業感が薄い（「押し付け」感がない）
- [ ] 季節感・タイムリー感が伝わる表現が自然に使われている

**禁止表現チェック**
- [ ] 「今すぐ申し込まないと」「期限が〜」などの焦らせ表現が過度でない
- [ ] 「他の塾はみんな〜」などの競合刺激がない
- [ ] ニュース性がなく「ただの営業」になっていない

---

## 共通品質チェック（全タイプ）

### 文字数・形式
- [ ] 件名：30文字以内（スマホで見切れない長さ）
- [ ] 本文：400-800文字（長すぎず短すぎず）
- [ ] 段落：3-5行ごとに改行（読みやすさ）
- [ ] 絵文字：件名に1個以内、本文は使用しない

### カスタマイズ項目確認
- [ ] `{name}` → 実際の宛先名に置換されている
- [ ] `{url}` → 実際の相手サイトURLに置換されている
- [ ] `□□□□` → 相手の具体的強みに置換されている
- [ ] `◯◯◯◯` → 具体的な改善ポイントに置換されている

### 法的・倫理的チェック
- [ ] 実績数字に根拠がある（でっち上げでない）
- [ ] 相手の商標・ブランド名を不適切に使用していない
- [ ] 反社会的勢力関連の企業・個人でないことを確認
- [ ] 個人情報（取得したURL以外）を不適切に使用していない
