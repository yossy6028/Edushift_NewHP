# PJ3 Xプロモ戦略（釘崎担当）

## 全体方針
Edushiftの知見を活かし、@yossyk14と@YSTConsultingの2アカウントで教育×AI領域のポジションを確立する。

## アカウント別戦略

### @yossyk14（個人ブランド）
- **テーマ**: 入試分析・国語学習Tips・教育トレンド
- **トーン**: 専門的だが親しみやすい、保護者目線
- **投稿頻度**: 1〜2回/日
- **強み**: 具体的な入試データ、実体験ベースの信頼感

### @YSTConsulting（法人ブランド）
- **テーマ**: アプリ紹介・AI活用体験・塾経営DX・HP改善
- **トーン**: プロフェッショナル×実践的、同業者目線
- **投稿頻度**: 1〜2回/日
- **強み**: アプリのDay紹介（毎日1本）、DX実装の具体ノウハウ
- **フォロワー現状**: 415 → 目標: 月+50〜100

## 競合分析からの反映ポイント
1. **価格訴求を避ける** → 競合は「HP制作○万円〜」で戦う → Edushiftは「知識・事例・成果」で差別化
2. **更新頻度で勝つ** → 競合の更新頻度は週1〜2 → こちらは毎日投稿で圧倒
3. **AI実装の具体性** → 他社は「AI導入支援」で抽象的 → Edushiftは実際のツール名・手順を公開
