# 設計: 第2の柱「塾DX・ツール開発」新設とLP/HP刷新

日付: 2026-07-03
ステータス: 承認済み

## 背景と目的

現状のEduShiftコンサルティングはHP販売が収益の中心。次の柱として
塾のバックオフィスアプリ開発・教材制作・指導サポートツール開発を前面に出す。
既存の自社開発SaaS「自動添削システム」(https://auto-tensaku-system.vercel.app/)
は開発力の証明としてLP上で簡潔に紹介する。

## 決定事項(ユーザー承認済み)

1. **柱の構成**: 3領域(バックオフィス/教材制作/指導サポート)を1本の新Pillar
   「塾DX・ツール開発」に統合。既存AI導入コンサル(Tool 01)は研修・導入支援と
   して残し住み分け・相互リンク。
2. **料金モデル**: 初期費+月額の2部料金(HP制作と同型)。
3. **価格帯**: 低価格入口型(初期3万〜30万、月額¥2,980〜¥9,800)。
4. **自動添削の紹介**: トップページの「開発実績」ミニセクション+新Pillarページ内
   の実例紹介の両方。
5. **トップページ**: ヒーローコピー含めて刷新(A案: まるごと伴走軸)。

## サービス定義

- slug: `dx-development` / label: 塾DX・ツール開発 / labelEn: School DX & Development
- category: `pillar` (Pillar · 03)
- 3領域:
  - バックオフィスアプリ: 指導報告書自動作成、保護者連絡自動化、請求・月謝管理、生徒DB化
  - 教材制作: オリジナル教材デジタル化、プリント自動生成、過去問データベース
  - 指導サポートツール: 自動添削、学習進捗ダッシュボード、宿題管理・小テスト自動化
