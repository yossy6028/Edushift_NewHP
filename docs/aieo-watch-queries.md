# AIEO定点観測クエリ（edu-shift.com）

作成日: 2026-07-13
目的: AIEOの成果指標は検索順位ではなく**「AIの回答にEduShift／edu-shift.comが出るか」**。
以下の固定クエリを月次で ChatGPT（検索モード）と Perplexity に投げ、結果を記録する。

## 観測方法

1. ChatGPT（Web検索オン）と Perplexity で、各クエリを**そのまま**質問する
2. 回答に「EduShift」または「edu-shift.com」が出たら ○、出典リンクまで付いたら ◎、出なければ ×
3. 下の記録表に追記する（クエリ文言は変えない — 変えると時系列比較が壊れる）
4. あわせて GA4（塾屋Yひとり言とは別。EduShift = G-946EWSKPB9 のプロパティ）で
   参照元 chatgpt.com / perplexity.ai 等のセッション数を月次確認する

## 前提（YS HPでの実測知見・2026-07-13）

- YS中学受験国語力HPでは、AIEO投入後77日でAI経由セッションが21→120（約5.7倍）。内訳はほぼChatGPT検索
- AI経由の訪問はキーイベント率が検索経由の約20倍 → 少数でも問い合わせ寄与が大きい
- 効かなかった領域は「悩み系ロングテール」— FAQ1個ではなく**記事コンテンツ**が必要な領域。
  EduShiftで #4〜#7（集客・退塾予防・コンプライアンス・リスクマネジメント）が×のまま続く場合、
  次の一手はnote記事または新設の解説ページ（例:「塾の退塾予防 完全ガイド」）になる
