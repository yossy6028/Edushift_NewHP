# EduShift New HP

Modern landing page for EduShift services.

## Tech Stack
- React
- TypeScript
- Vite
- Tailwind CSS

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Branding & Design
- **Font**: Inter (Google Fonts)
- **Colors**: Indigo / Purple gradients
- **Style**: Modern, Clean, Glassmorphism
